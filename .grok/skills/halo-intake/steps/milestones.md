# Milestone strategy

Propose **default** dependency-ordered milestones (usually 3–5). Each must deliver **visible** user value.

Also offer:

- Fewer/bigger
- More/smaller

Each milestone: name, one-liner scope, depends_on.

AskUserQuestion. Then confirm names.

Lock: `intake.milestones[]` = `{n, slug, name, scope, done_when_draft, depends_on[]}`.
